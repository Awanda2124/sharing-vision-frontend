# Frontend Test - Sharing Vision Fullstack Engineer

Dashboard untuk mengelola artikel (Post Article): Published, Drafts, Trashed, tambah artikel, edit artikel, dan preview blog.

## Tech Stack

HTML, CSS, JavaScript (Vanilla), Bootstrap/Tailwind CSS

## Cara Running

1. Clone repository:
```bash
   git clone https://github.com/username/frontend-repo.git
   cd frontend-repo
```

2. Buka `index.html` langsung di browser, **atau** jalankan local server (disarankan biar fetch API ke backend tidak terblokir):
```bash
   npx live-server --port=3000
```
   Lalu buka `http://localhost:3000`

3. Pastikan backend sudah berjalan (lihat repo backend), karena frontend ini butuh koneksi ke backend API.
